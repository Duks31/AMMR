Downloaded the datasets and placed them in the data/ directory.

Splitted the dataset into recyclable and non-recyclable categories.
- recyclable
    - cardboard
    - plastic
    - paper
    - glass
    - metal
- non-recyclable
    - trash

Dataloaders created using Pytorch's DataLoader class.
- Used torchvision.transforms for image transformations.
    - Resized images to 224x224. To match the input size of pre-trained models. To reduce computational load. To maintain consistency across the dataset.
    - Converted images to tensors.
    - Normalized images using mean and std deviation of ImageNet dataset. This is because we will be using pre-trained models trained on ImageNet. if not normalized, the models activations can become unbalanced, leading to poor accuracy. 
- Used random_split to create training and validation sets.

Dataset Imbalance:
- Recycle: Non-Recycle = 17:1 (2390 vs 137)
- First attempt: weighted loss(assignment of higher weights to minority class during loss calculation)
